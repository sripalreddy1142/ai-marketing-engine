\# AI Marketing Intelligence Engine



\## 🚀 Project Overview



AI Marketing Intelligence Engine is a web application that generates marketing content using AI. Users can input a campaign topic and select content type to generate marketing materials automatically.



\## ✨ Features



\- AI Content Generation

\- LinkedIn Post Generator

\- Instagram Caption Generator

\- Email Content Generator

\- Content Type Selection

\- FastAPI Backend

\- Simple Responsive UI



\## 🛠 Tech Stack



\### Frontend

\- HTML

\- CSS

\- JavaScript



\### Backend

\- FastAPI (Python)



\### AI Model

\- Groq (Llama 3.1)

\## ⚙️ How to Run Locally



\### Clone Repository



git clone https://github.com/sripalreddy1142/ai-marketing-engine.git







\### Backend Setup



cd backend

python -m venv venv

venv\\Scripts\\activate

pip install fastapi uvicorn groq





\### Run Backend





uvicorn main:app --reload





\### Run Frontend



Open:





frontend/index.html
